# Reference Notebooks

These notebooks were used to create the initial versions of tables that are now maintained by the E2E pipeline. They are not actively run, but are kept here as reference for understanding how the tables were originally built.

## Files

- **CreateRawAffiliationStringsInstitutionsMV.ipynb** — Initial creation of the raw affiliation strings institutions materialized view
- **CreateWorkAuthorAffiliationsMV.ipynb** — Initial creation of the work-author-affiliations materialized view
- **CreateWorkAuthors.ipynb** — Initial creation of the work authors table
- **CreateWorksLegacyTables.ipynb** — Initial creation of the works legacy tables
- **CreateWorkAuthorAffiliations.ipynb** — Initial creation of the work-author-affiliations table
