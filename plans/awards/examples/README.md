# Funder-intelligence examples

Worked examples of what you can do with OpenAlex's open funding metadata — assembling a funder's
research footprint, measuring its impact, and stress-testing the methods. Each example is a
self-contained, reproducible project (scripts + data + figures + write-up).

## Examples

| Example | What it shows |
|---|---|
| [`tula-hakai-funder-impact/`](tula-hakai-funder-impact/) | End-to-end impact case study of the Tula Foundation / Hakai Institute: multi-strategy corpus assembly with provenance, LLM verification, precision/recall against the funder's own list, citation & FWCI impact, downstream global reach, cross-institution collaboration, co-funding, and field specialization. **Published:** [blog.openalex.org](https://blog.openalex.org/the-hakai-institute-as-seen-by-openalex/) |
| [`aha-oa-compliance/`](aha-oa-compliance/) | Open-access **compliance** dashboard for the American Heart Association — live OpenAlex data, interactive charts, and per-work compliance flags against AHA's 12-month PubMed Central deposit policy. Deploy target: `openalex.org/examples/AHA-dashboard`. |

More to come — landscape analyses, gap analyses, co-funding networks, and other funder use cases.
